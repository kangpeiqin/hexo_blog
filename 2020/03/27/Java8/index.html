Lambda表达式

Lambda表达式返回的是函数接口类型，其实与匿名函数的作用是相同的。

JDK提供函数接口

JDK中提供了很多实用的函数接口，大概列出了一些，具体可看下API文档,在使用过程看下源码可以理解的更透彻。Stream API 中很多方法使用了这些函数接口。Predicates：断言型、Functions：接受一个参数并生成结果、Suppliers：产生给定泛型类型的结果、Consumers：表示要对单个输入参数执行的操作。可以从源码入手了解这些接口，可以尝试写几个示例了解如何使用，再过来深入理解，其实在Stream API中大量使用了这些接口,在使用Stream API的时候就不知不觉用到了。
